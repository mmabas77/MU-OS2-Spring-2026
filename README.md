# OS2 — Spring 2026

Operating Systems 2 covers advanced OS concepts — memory management, file systems, protection, and security — alongside practical mobile development with Flutter and containerization with Docker.

**Lecturers:** Dr. Mohamed Hamdousa · Dr. Waleed Mohamed
